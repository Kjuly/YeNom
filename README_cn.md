YeNom
=====

YeNom - 一个iOS上简单方便的记帐工具。

# 介绍

一个iOS上简单方便的记帐工具。

[__进入应用商店__](https://itunes.apple.com/cn/app/yenom/id1120637689?ls=1&mt=8)

# 操作说明

__主界面__：

  - 向下/上拖动记录列表后停留一秒（延迟时间可在 “设置－通用－延迟时间” 里更改），打开收入（支出）的记录设置页面；
  - 可直接在应用图标上使用 3D Touch 激活快速记录添加按钮。

__记录页面的数字键盘__：

  - 删除：在键盘或数字区域向右滑动；
  - 恢复：在键盘或数字区域向左滑动；
  - 计算：内置简易计算功能。

__标签界面__：

  - 向下拖动标签列表添加新标签；
  - 在记录页面轻击标签区域，将显示所有标签；
  - 在所有标签页面，再次轻击标签区域，将显示标签搜索页面；
  - 轻击未使用过的标签左侧“+”按钮，将为该标签转化为类别，可以将相关标签放于同一类别下；
  - 在“设置－通用－标签链接”中，可以为标签设置链接；
  - 标签报表可以通过轻击报表中的饼图现实。

__报表__：

  - 轻击记录列表任一月份的区域，可以现实相应月份的报表，向上／下滑动可切换到下／上一月份的报表；
  - 在主菜单中的报表中，可以浏览一定时间范围内的报表。

__数据管理__：

  - __设置－本地数据管理－导入__：可以通过 API 将外部 CSV 格式的数据导入到应用中（具体文档后续会被添加）；
  - __设置－本地数据管理－导出__：可以将本地数据导出为 CSV 或 PDF 格式；
  - __设置－数据同步－同步__：可以启用 iCloud 同步以在多台设备间同步；
  - 所有照片都存储于一个专门的相册（“__YeNom__”）中。
