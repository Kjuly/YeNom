Usage

Root View:

  • Drag the table view down (up) and hold a second to open the record setting view for income (expenses);
  • You can change the time in settings view, and there're three options (0.5s, 1.0s, 1.5s) available.

Tag View:

  • Drag the table view down to add a new tag. Data Management:
  • You can clean all records or tags in settings view's "Data Management" section;
  • Photos are stored in Albums with the name of "YeNom Photo Lib";
    Note: Photo related features haven't managed very well right now, sorry for that!
